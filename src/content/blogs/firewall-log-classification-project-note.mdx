## Overview

Firewall logs contain useful signals about network activity, but the raw data
can be noisy, high-volume, and difficult to inspect manually. The goal of this
project was to classify internet firewall log records into multiple classes
using machine learning.

## What The Work Explored

- Preparing firewall log data for classification
- Comparing machine learning models for multiclass prediction
- Evaluating model performance across traffic/action classes
- Understanding how structured security logs can support automated analysis

## Publication

This work was published as:

[Machine Learning Approach on Multiclass Classification of Internet Firewall Log Files](/publications/machine-learning-firewall-log-classification)

## Takeaway

Security data is rarely clean, but with careful preprocessing and evaluation,
machine learning can help turn large log streams into actionable structure.
